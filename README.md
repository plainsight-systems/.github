# .github

This repository holds the [GitHub organisation profile](https://github.com/plainsight-systems) for **Plainsight Systems LLC**.

The org-profile content rendered at <https://github.com/plainsight-systems> lives at [`profile/README.md`](profile/README.md). GitHub renders that file automatically as the organisation's landing page; no further configuration is required.

## Editing the profile

1. Edit `profile/README.md`.
2. Commit and push to `main`.
3. The change is live at github.com/plainsight-systems within a minute.
